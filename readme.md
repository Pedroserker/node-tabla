Aplicación de consola usando node.

```
Opciones:
      --help     Muestra ayuda                                        [booleano]
      --version  Muestra número de versión                            [booleano]
  -b, --base     La base de la tabla de multiplicación      [número] [requerido]
  -s, --show     muestra la tabla en consola         [booleano] [defecto: false]
  -l, --limite   limite de la tabla                       [número] [defecto: 10]
```
